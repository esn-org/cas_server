-- SUMMARY --

The CAS Server module turns a Drupal install into the central
authentication system. It makes your database the primary location
for other systems to use for authentication in a SSO environment.

This module tries to abide by the CAS protocol:
  http://www.jasig.org/cas/protocol

-- REQUIREMENTS --

None.


-- INSTALLATION --

* Install as usual, see http://drupal.org/node/70151 for further information.


-- CONFIGURATION --

* Customize the settings in Administer >> Site configuration >>
  CAS Server.
